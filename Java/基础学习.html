# 2.cmdеёёи§Ғе‘Ҫд»Ө

# 3.зҺҜеўғеҸҳйҮҸ

# 3.1дҪңз”Ёпјҡ

   еҰӮжһңжҲ‘жғіиҰҒеңЁCMDзҡ„д»»ж„Ҹзӣ®еҪ•дёӢпјҢйғҪеҸҜд»ҘеҗҜеҠЁжҹҗдёҖдёӘиҪҜд»¶пјҢйӮЈд№Ҳе°ұеҸҜд»ҘжҠҠиҝҷдёӘиҪҜд»¶зҡ„и·Ҝеҫ„й…ҚзҪ®еҲ°зҺҜеўғеҸҳйҮҸдёӯзҡ„PATHйҮҢйқўгҖӮ
   еңЁеҗҜеҠЁиҪҜд»¶зҡ„ж—¶еҖҷпјҢж“ҚдҪңзі»з»ҹдјҡе…ҲеңЁеҪ“еүҚи·Ҝеҫ„дёӢжүҫпјҢеҰӮжһңеңЁеҪ“еүҚеҪ•иҜҫжІЎжңүеҶҚеҲ°зҺҜеўғеҸҳйҮҸзҡ„и·Ҝеҫ„дёӯеҺ»жүҫгҖӮеҰӮжһңйғҪжүҫдёҚеҲ°е°ұжҸҗзӨәж— жі•еҗҜеҠЁгҖӮ

# 3.2жӯҘйӘӨпјҡ

1.еҸій”®жҲ‘зҡ„з”өи„‘пјҢйҖүжӢ©еұһжҖ§гҖӮ
2.зӮ№еҮ»е·Ұдҫ§зҡ„й«ҳзә§зі»з»ҹи®ҫзҪ®
3.йҖүжӢ©й«ҳзә§пјҢеҶҚзӮ№еҮ»дёӢйқўзҡ„зҺҜеўғеҸҳйҮҸгҖӮ
4.жүҫзі»з»ҹеҸҳйҮҸйҮҢйқўзҡ„PATH
5.жҠҠиҪҜд»¶зҡ„е®Ңж•ҙи·Ҝеҫ„пјҢй…ҚзҪ®еҲ°PATHеҪ“дёӯе°ұеҸҜд»ҘдәҶгҖӮ
6.пјҲеҸҜеҒҡеҸҜдёҚеҒҡпјүе°ұжҳҜжҠҠиҮӘе·ұй…ҚзҪ®зҡ„и·Ҝеҫ„пјҢз§»еҠЁеҲ°жңҖдёҠйқўгҖӮ
жіЁж„Ҹпјҡ
з§»еҠЁзҡ„еҘҪеӨ„пјҡеңЁCMDдёӯжү“ејҖиҪҜд»¶ж—¶пјҢдјҡе…ҲжүҫеҪ“еүҚи·Ҝеҫ„пјҢеҶҚжүҫзҺҜеўғеҸҳйҮҸпјҢеңЁзҺҜеўғеҸҳйҮҸдёӯжҳҜд»ҺдёҠеҫҖдёӢдҫқж¬ЎжҹҘжүҫзҡ„пјҢеҰӮжһңи·Ҝеҫ„ж”ҫеңЁжңҖдёҠйқўжҹҘжүҫзҡ„йҖҹеәҰжҜ”иҫғеҝ«гҖӮ
